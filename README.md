Assignment 3 – DOM Implementation
📌 About
A simple web page built with HTML, CSS, and JavaScript that demonstrates DOM manipulation.
✨ Features

Greeting header that updates with the user's name
Four colour boxes (Red, Blue, Green, Yellow) that fill with colour on click
Input field and button to personalise the greeting

🚀 How to Run

Download all files into the same folder:

index.html
style.css
script.js


Open index.html in any web browser
No installation needed!

├── style.css    → Styling & layout

🛠️ Technologies Used

HTML5

Vanilla JavaScript (DOM manipulation, event delegation)
